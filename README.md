# Walmart Analysis Dashboard

## Overview
An interactive Power BI dashboard designed to analyze Walmart sales performance. The dashboard provides insights into sales trends, profit, customer behavior, product categories, and regional performance.

## Features
- Sales & Profit KPIs
- Monthly Sales Trend
- Regional Performance Analysis
- Product Category Analysis
- Customer Segment Analysis
- Interactive Slicers

## Tools
- Power BI
- Power Query
- DAX

## Dataset
Walmart Sales Dataset
